# Getting Started with React

Reactの始め方について解説します。

## Reactとは

ReactはFacebook（現Meta）が開発したJavaScriptライブラリです。

## 特徴

- コンポーネントベース
- 仮想DOM
- 宣言的UI

## 簡単な例

```tsx
function App() {
  return (
    <div>
      <h1>Hello React!</h1>
    </div>
  )
}
```
